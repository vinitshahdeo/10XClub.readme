## Resources Presented During the Workshop

### **Badges for GitHub Repositories**
Add engaging and informative badges to your GitHub repositories using these tools:  

- **[Shields](https://shields.io/):** Create customizable badges for your GitHub profile.
- **[Reporoster](https://reporoster.com/):** Shout-out supporters in your GitHub `README`.
- **[OSS Insight Widgets](https://next.ossinsight.io/widgets):** Visualize insightful data about your open-source project.
- **[LeetCode Badge](https://leetcode-badge.vercel.app/):** Display your LeetCode progress and achievements.
- **[Visitors Badge](https://visitorbadge.io/):** Track and display the number of visitors to your repository.
- **[README Stats](https://github.com/anuraghazra/github-readme-stats):** Show dynamic stats like top languages, stars, and more in your repository `README`.

### **Resume Templates**
Use these professional templates to create standout resumes:  

- **[FAANGPath Template](https://www.overleaf.com/latex/templates/faangpath-simple-template/npsfpdqnxmbc):** A clean and minimal resume template optimized for tech roles.  
- **[Awesome CV Template](https://www.overleaf.com/latex/templates/awesome-cv/dfnvtnhzhhbm):** A feature-rich, visually appealing LaTeX template for crafting an impressive resume.
